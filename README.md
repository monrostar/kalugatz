# kalugatz
==================

Requirements
------------

- PHP 7+

Building Project
---------------------

Clone repository: `https://github.com/monrostar/kalugatz.git`

```sh
$ git clone https://github.com/monrostar/kalugatz.git
$ composer install
```

Use
---------------------------

```sh
$ php index.php string=<string>
```

Tests
---------------------------

```sh
$ php .\vendor\phpunit\phpunit\phpunit src\Test\SorterTest.php
```
